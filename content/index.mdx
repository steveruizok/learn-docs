## Installation

You can install the
[Learn Design System](https://store.framer.com/package/steveruizok/education)
package through the Framer X Store.

1. Open a Framer X project.
2. Click the Store tab to open the Framer X Store.
3. Search for "Learn". Click on the Learn Design System link.
4. Click the Install button.
5. When the install completes, open the components list in your project to see
   all of the package's components.

## Source and Examples

The Learn Design System is open source. You can find source files for the Learn
Design System on its
[Github repository](https://github.com/framer/framer-education) in the **Learn**
folder. The repository also includes several examples in the **Projects**
folder.

1. Visit the [Github repository](https://github.com/framer/framer-education).
2. Click the green Download button.
3. Unzip the download and browse the files.
